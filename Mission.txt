Goal: Create a simplified version of a database for phonebook using go.

The database will simply be a textfile. The text in this file will be formatted in such a way that it can be read by some convention.

Data:
- ID (Primary key)
- First Name
- Last Name
- Address
- Phone Number
- Occupation

Actions:
1. Add to database
2. Get all records
3. Find by ID
4. Update a record
5. Delete a record
6. Search/Filter (Criteria TBD)
